Notes for Javascript

Where to

In html, Javascript must be inserted between <script> and </script> tags.
A Javascript function is a block of JavaScript code, that can be executed when "asked" for.
Scripts can be placed in the <body> or <head> section of an HTML page, or in both. Note: keeping all code in one place is a good habit.
It is a good idea to place scripts at the bottom of the <body> element. This can improve page load, because HTML display is not blocked by scripts loading.
Scripts can also be placed in external files. External scripts are practical when the same code is used in many differnt web pages. Javascript files have the file extension .js. To use an external scruot, put the name of the script file in the src attribute of the <script> tag. External scripts cannot contain <script> tags.
Placing JavaScripts in external files has some advantages:

  • It seperates HTML and code.
  • It makes HTML and JavaScript easier to read and maintain.
  • Cached JavaScript files can speed up page loads.

Output

JavaScript can "display" data in different ways:

  • writng into an alert box, using window.alert().
  • writing into the HTML output using document.write(). Note: using this after an HTML document is fully loaded will delee all existing HTML. This should be only used for testing.
  • writing into an HTML element using innerHTML. Note: this is a commonly used method.
  • writing into the browser console, using console.log. Note: to access an HTML element, JavaScript can use the document.getElementById(id) method. The id attribute idenifies the HTML element. The innerHTML property is the element content.

Note: Activate debugging in our browser (Chrome, IE, Firefox) with F12, and select Console in the debugger menu to see the console log.
